This is a Json Beuatifier.
Across the top left corner are the show help toggle, hide left & and right sidebars toggle, reset saved data button, and undo reset saved data buttons.
Along the left sidebar are the sessions. You can create (+), select, trasg (x) and rename (doube click) sessions.
Acrosss the top are the tabs for the currently active session. You can create (+), select, trasg (x) and rename (doube click) tabs. A new tab will be automatically created if the last tab is used.
Along the right sidebar are the trashed tabs for the currently selected session. You can preview (hover) or restore (click) trashed tabs. To permanently delete all trashed tabs for the currently selected session, click the trashcan icon at the top of the right sidebar.
Note, there is currently no way to retrieve deleted sessions.
In left giant box, input the unformatted Json(s). This supports comments, multiple Json objects, and some invalid Json tolerance.
In the right giant box, the formatted Json will output. Click to select the entire output, or drag select to select a subset.
At the bottom will be a 1-line of the Json output. Click to select the entire output, or drag select to select a subset.
